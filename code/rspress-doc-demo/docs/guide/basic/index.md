# 基础用法


## 安装

```shell

mkdir rspress-doc-demo && cd rspress-doc-demo  //创建根目录
npm init -y                                    //初始化package.json
yarn add rspress typescript ts-node -D         //安装依赖

```

## 新建主目录

```js
mkdir docs && echo '# Hello Repress' > docs/index.md
```

## 配置顶部右侧导航

```js
  touch _meta.json
```
